jQuery Selectors

Please use the browser console to play with jQuery

Here is some example code:

Selects the h1 tag and displays the h1 tag element.

var h1 = $("h1");

h1[0]


Selects the li tag, loops through all li tags and displays it.

var lis = $("li");

for(var i = 0; i < lis.length; i++) {

  console.log(lis[i]);

}
